- Used to iterate a piece of code
```
# To Print 1 to 5 using For loop

for (let i=1; i<=5; i++){
	console.log(i);
}
```

- Print all odd Numbers (1 to 15)
```
for(let i=1; i<=15; i++){
    if(i%2!=0){
        console.log(i);
    }
}
```

- Print all even numbers (2 to 10)
```
for(let i=2; i<=10; i++){
    if(i%2==0){
        console.log(i);
    }
}
```
